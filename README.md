# Monkey Programming Language Interpreter written in OCaml

[Writing an Interpreter in Go](https://interpreterbook.com/) in OCaml.

## Environments

- ocaml `4.07.0`
- opam `2.0.0-rc2`
- dune `1.1.1`

## Build

```
dune build
```

## Run

```
dune exec monkey
```

## Test

```
dune runtest
```
